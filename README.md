# Development and Testing of a Comprehensive Incident Response Plan

This graduation project focuses on designing, developing, and testing a comprehensive
Cyber Security Incident Response Plan based on industry best practices.

## Project Objectives
- Develop a structured Incident Response Plan
- Define roles and responsibilities during incidents
- Simulate real cyber security incidents
- Test and evaluate the effectiveness of the response plan
- Document findings and lessons learned

## Project Phases
1. Research and Planning
2. Incident Response Plan Development
3. Incident Simulation and Testing
4. Final Reporting and Documentation

## Methodology
The project follows the Incident Response lifecycle:
- Preparation
- Detection and Analysis
- Containment
- Eradication
- Recovery
- Lessons Learned

## Tools & Technologies
- SIEM (ELK Stack)
- Windows Server & Windows 10
- Kali Linux
- Sysmon & Windows Event Logs
- Virtualization Technologies
